# Change Log
All notable changes of my website will be documented here.

This project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

---
## [2.0.1] - 2024-12-15
### Features
- Subscribe Popup
- Added Skip to Content Feature
- Added the Christmas theme
### Improvements
- Colorful Logo on Mouse Over
- New Landing Page Button Colors
- Other improvements

## [2.0.0] - 2023-03-04 (New Look)
### Features
- Added Dark Mode support
### Improvements
- Updated and improved the UI
- Changed Loader Animation
- Made Color Contrast Ratio Better
- Converted Logo From PNG to SVG
- Changed Lightbox From Jquery Lightbox to FSLightbox (Pure JavaScript)
- Updated Apple Touch Icon and Added Android Icon
- Added `manifest.json`
- Changed Caret Color
- Other improvements
### Fixes
- Fixed Some Spelling Errors

## [1.5.0] - 2022-12-04
### Features
- Added Loader, Background Overlay, Lazy Loading
- Added Theme Color and Apple Touch Icon
- Added Downloads Page Link To the Main Website
### Improvements
- Made the copyright part of the Footer dynamic
- Converted Some Images To WEBP
- Replaced JQuery with Pure JavaScript
- Minified Entire Website Code
- Optimizations and Minifications for clarity
### Deletions
- Removed Other `[error-code].html` pages except the 404 page

## [1.0.0] - 2021-01-15 (Rebirth)
### Initial Features
- Reintroduced site header and footer with a new design
- Added homepage with sections: About, Services, Works, and Contact
- Added 404 page and tested other error type pages
- Initialized using JQuery and implemented LightBox
- Added a Button to Go Top
### Improvements
- Made Every Service Clickable
- Compressed Some Images
- Minifications, Image compressions, and Optimizations
- Updated Design of a 404 Page and a `<noscript>` detection

## [0.5.1] - 2020-12-03
### Highlights
- Final stable version of the original v1 (now labeled v0.5 – Prototype version)
- Website was deleted on 2020-12-23 due to lack of optimization and negative feedback
### Features
- Added `<noscript>` detection for users with JavaScript disabled
- Added 404 Error Page
- Added loading screen
### Fixes
- General layout improvements
- Minor styling consistency fixes

## [0.5.0] - 2020-11-15
### Initial Features
- Introduced site header and footer
- Added homepage, portfolio/works page, and about page
- Gallery page added (non-optimized images)
- Contact form via Google Forms
- Used jQuery for interactions
- First official version of the site
